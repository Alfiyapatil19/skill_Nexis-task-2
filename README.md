# 🤖 Skill Nexis – Machine Learning & AI (Week 2)

## 📌 Project Overview

This repository contains the solutions for **Week 2** of the **Skill Nexis Machine Learning & AI Internship**. The focus of this week is **Supervised Learning**, covering both **Regression** and **Classification** techniques.

The repository includes two assignments and one mini project that demonstrate model building, training, prediction, and evaluation using Python and Scikit-learn.

---

## 📚 Topics Covered

* Linear Regression
* Logistic Regression
* Decision Tree
* Random Forest
* Model Evaluation Metrics

  * Mean Squared Error (MSE)
  * R² Score
  * Confusion Matrix

---

## 📝 Assignment 1: House Price Prediction using Linear Regression

### Objective

Build a Linear Regression model to predict house prices using a housing dataset.

### Tasks

* Load and preprocess the housing dataset.
* Split the dataset into training and testing sets.
* Train a Linear Regression model.
* Predict house prices.
* Evaluate the model using MSE and R² Score.
* Visualize Actual vs Predicted house prices.

---

## 📝 Assignment 2: Titanic Survival Prediction using Logistic Regression

### Objective

Develop a Logistic Regression model to predict passenger survival on the Titanic.

### Tasks

* Load and preprocess the Titanic dataset.
* Handle missing values.
* Encode categorical variables.
* Split the dataset into training and testing sets.
* Train a Logistic Regression model.
* Predict survival outcomes.
* Evaluate the model using:

  * Accuracy Score
  * Confusion Matrix
  * Classification Report

---

## 🚀 Mini Project 2: House Price Prediction Model

### Objective

Build a machine learning model that predicts house prices using Linear Regression.

### Tasks

* Train a Linear Regression model.
* Predict house prices.
* Evaluate the model using the R² Score.
* Plot Predicted vs Actual house prices.
